Random Bee Movie

What is this?

It's just a site that pulls 3 lines at random from the Bee Movie script and pairs them with random pictures from the movie.

Why?

I really don't know. But I made it and it's here

How?

I wrote it in Node.js. It uses a function with a pretty simple Regex to pull sentences from a file with the script. A similar function pulls pictures at random to pair them with. The source is on my GitHub if you're really interested.
